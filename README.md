NLP Project - Full Transformer Architecture from Scratch (Backend)

Designed and implemented a custom, complete Transformer model architecture from the ground up, optimized for English-to-Spanish translation tasks.

## Key Components
- **Backend & ML Architecture:** Full Encoder-Decoder framework completely from scratch, featuring custom Multi-Head Attention mechanisms, Positional Encoding layers, and a dedicated Byte Pair Encoding (BPE) tokenization pipeline.
- **API & Backend Integration:** Built using Python and Flask (`app.py`), creating a clean, high-performance backend service to manage model inference requests, preprocessing pipelines, and translation routing.

**Technologies Used:** Python, PyTorch, Flask, REST APIs.
